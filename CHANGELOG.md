## [1.0.0] - 2024-12-01

### Added

- Initial release of claude-prompts-library
- Coding prompts: code review, debugging, refactoring
- Writing prompts: technical documentation, blog posts
- Analysis prompts: data analysis, security audit
- DevOps prompts: incident response, runbook generation
- Templates: system prompt template, few-shot prompt template
- Comprehensive README with architecture diagram

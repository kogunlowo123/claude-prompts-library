# Blog Post Prompt

## Purpose

Create engaging, informative technical blog posts that explain concepts clearly and provide practical value to readers.

## System Prompt

```
You are a skilled technical blogger who writes engaging content that balances depth with accessibility. You use clear explanations, practical examples, and a conversational yet professional tone. You structure posts for both skimming and deep reading.
```

## Prompt Template

```
Write a technical blog post about:

**Topic:** {topic}
**Target Audience:** {audience}
**Goal:** {goal}  (educate / compare / announce / tutorial)
**Word Count:** {word_count}
**Key Points to Cover:**
{key_points}

**Requirements:**
- Engaging opening that hooks the reader
- Clear section headers for scannability
- At least one practical code example or diagram
- Real-world use case or scenario
- Conclusion with actionable takeaways
- SEO-friendly title and meta description

**Tone:** {tone}  (conversational / formal / opinionated)
```

## Variations

### Comparison Post
```
Write a comparison blog post: {technology_a} vs {technology_b}

Cover: key differences, performance benchmarks, use case fit,
migration considerations, and a recommendation matrix.
```

### Tutorial Post
```
Write a step-by-step tutorial blog post for: {task}

Include: prerequisites, environment setup, step-by-step instructions
with code, expected output at each step, troubleshooting tips.
```

### Deep Dive
```
Write a deep-dive blog post explaining: {concept}

Start with the basics, then progressively reveal complexity.
Include diagrams, code examples, and links to further reading.
```

## Structure Template

```
1. Title (compelling, specific)
2. Introduction (problem statement, why it matters)
3. Background (brief context, no more than 2 paragraphs)
4. Main Content (3-5 sections with headers)
5. Practical Example (code or walkthrough)
6. Conclusion (summary, next steps, call to action)
```
